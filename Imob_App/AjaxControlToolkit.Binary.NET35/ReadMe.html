AJAX Control Toolkit — May 2012 Release

This release of the Ajax Control Toolkit introduces a new AjaxFileUpload control. This control supports several highly requested features including:

  • File Upload Progress - A progress bar is displayed while a file is being uploaded.
  • Multiple File Upload - Upload multiple files at a time.
  • Drag-and-Drop Interface - Drag files onto the control to upload the files.

This release of the Ajax Control Toolkit also includes important bug fixes and feature enhancements for the HtmlEditorExtender control including:

  • Source View - Enter HTML content by switching between Source and Design view.
  • Image Upload - Upload images by using the Insert Image button.

This release has been updated to use the latest version of the Microsoft Anti-Cross Site Scripting Library (version v4.2).
